Basic React app

This React app demonstrates how to make frontend calls with GraphQL API queries to an Adobe Commerce backend.

It uses the following product-based queries to fetch data that drives a simple products page that might exist on a larger storefront:

  • products
  • categories

In addition, this app:

  • Connects to a PWA Studio-based Adobe Commerce on cloud infrastructure project.
  • Does not require authentication.
  • Does not support GraphQL mutations.

To learn more about this app, see the GitHub repository.
